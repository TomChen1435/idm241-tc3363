# Alpha

## Triggers

- Product Card
    - Hover on
    - Hover off

## Rules & Feedbacks

- Product Card
    - Hover on
        - The cursor changes to a pointer
        - The image zooms in slightly
        - All texts become underlined
    - Hover off 
        - The cursor changes to an arrow
        - The image zooms out slightly
        - All text decorations are removed

## Loops & Modes

- None
